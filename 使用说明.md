# RightMenu 使用说明

## 📖 简介

RightMenu 是一款为 macOS Finder 添加右键菜单增强功能的工具，让您的工作效率倍增！

## ✨ 功能特点

- **快速新建文件**：右键菜单直接创建各种类型的文件
  - 文本文档 (.txt)
  - Word 文档 (.docx)
  - Excel 表格 (.xlsx)
  - PPT 演示文稿 (.pptx)
  - Markdown 文件 (.md)

- **实用工具**：
  - 📋 复制路径：一键复制文件/文件夹的完整路径
  - 💻 在终端打开：快速在指定目录下打开终端

- **全局可用**：在用户目录和外部磁盘的任何位置都可使用

## 🚀 安装步骤

### 1. 安装应用

1. 打开 `RightMenu_Installer.dmg`
2. 将 `RightMenu.app` 拖动到 **应用程序** 文件夹
3. **首次打开**：右键点击应用 → 选择 "**打开**"
4. 在弹出的安全警告中点击 "**打开**"

### 2. 启用 Finder 扩展

1. 打开 **系统设置**（System Settings）
2. 进入 **隐私与安全性** → **扩展**（Extensions）
3. 点击 **Finder 扩展**（Finder Extensions）
4. 勾选 ✅ **RightMenu**

### 3. 重启 Finder（可选）

如果菜单没有立即出现，可以重启 Finder：
- 按住 **Option** 键
- 右键点击 Dock 上的 Finder 图标
- 选择 "**重新启动**"

## 📝 使用方法

### 新建文件

1. 在 Finder 的**空白处**或**文件夹**上右键
2. 选择 "**新建...**" → 选择文件类型
3. 文件将在当前目录下创建

### 复制路径

1. 右键点击任意文件或文件夹
2. 选择 "**复制路径**"
3. 路径已复制到剪贴板，可直接粘贴使用

### 在终端打开

1. 右键点击文件夹（或文件夹内空白处）
2. 选择 "**在终端打开**"
3. 终端会自动打开并切换到该目录

## ⚙️ 自定义设置

1. 启动 **RightMenu** 主应用
2. 在设置界面中：
   - 勾选/取消勾选想要显示的菜单项
   - 更改会立即生效

## ❓ 常见问题

### Q1: 右键菜单没有出现？

**解决方案：**
1. 确认已在"系统设置 → 扩展 → Finder 扩展"中启用
2. 重启 Finder（参考安装步骤第3条）
3. 重启电脑

### Q2: 首次打开提示"无法打开"？

**解决方案：**
1. **不要**直接双击打开
2. **右键**点击应用 → 选择 "**打开**"
3. 在弹出的对话框中点击 "**打开**"

这是 macOS 对未签名应用的安全机制，只需首次这样操作即可。

### Q3: 新建的 Office 文件无法打开？

**说明：**
- `.txt` 可以用任何文本编辑器打开
- `.docx`、`.xlsx`、`.pptx` 是空白模板，需要用 Microsoft Office 或 WPS 打开

### Q4: 支持哪些 macOS 版本？

- 最低要求：macOS 11.0 (Big Sur)
- 推荐：macOS 12.0 (Monterey) 及以上

## 🗑️ 卸载方法

1. 打开 **系统设置** → **扩展** → **Finder 扩展**
2. 取消勾选 **RightMenu**
3. 将 `RightMenu.app` 拖到废纸篓
4. 重启 Finder
