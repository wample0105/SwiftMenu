# 更新日志 (CHANGELOG)

## v1.2.0 (2026-07-20)

### 🌿 轻量化与性能
- 移除扩展心跳文件、轮询检查、`pluginkit` 强制复活和主程序常驻机制，空闲状态不再产生周期性磁盘 I/O。
- 设置应用改为按需运行，关闭最后一个窗口后自动退出；自动迁移并注销旧版本登录项。
- 菜单配置改为单次快照读取并缓存 SF Symbols，减少右键菜单热路径中的重复工作。
- Finder 菜单覆盖用户主目录及 `/Volumes` 下的本地、外接和网络挂载卷。

### 🛡 可靠性
- 新建文件、粘贴冲突预检、文件复制与移动改到串行后台队列，避免慢速网络盘或大文件操作阻塞 Finder 扩展主线程。
- 普通复制改为隐藏暂存后原子落位；跨卷移动完成安全复制后才删除源文件，降低中断后留下可见半成品的风险。
- 移除冲突弹窗中的 semaphore 死锁路径，增加取消、跳过、保留两者和安全替换。
- 替换操作先复制到同目录暂存位置，成功后再原子替换，避免先删除目标导致数据丢失。
- 修复首次启动默认开关全部关闭、菜单顺序缺失或含无效值的问题。
- 新建 Word、Excel、PowerPoint 改为最小合法 OOXML 文档，不再创建无法打开的零字节伪文档。
- 修复 Finder 右键“新建文件”和粘贴目标目录被沙盒拒绝的问题。Finder Sync 的 `targetedURL` 只提供路径，不会像打开面板那样授予 Powerbox 权限；Developer ID 直装版因此显式允许 Home 与 `/Volumes/` 目录读写，并继续使用安全书签延续跨菜单动作的源文件权限。
- 修复“在终端打开”因 Finder Sync 沙盒导致的 Terminal 启动失败和目录无权限错误；改用 macOS 自带的 “New Terminal at Folder” 系统服务，只传递路径文本，不申请 AppleScript 自动化权限。
- 修复复制或剪切后再粘贴时源文件权限失效的问题；复制/剪切动作将临时 Finder 权限转换为安全书签，粘贴时恢复源文件和目标目录的 security scope 后再交给后台队列。
- 修复 SwiftMenu 复制或剪切后，在新目录右键看不到“粘贴”的问题；菜单现在同时识别系统文件 URL 与 SwiftMenu 安全书签剪贴板类型，不再因沙盒隐藏 `public.file-url` 而误判为空剪贴板。
- 新增 24 项核心逻辑回归测试、真实 App Sandbox 安全书签移动探针、LibreOffice 实际解析、菜单构造性能门禁和 macOS 12 严格并发类型检查。

### 📦 生产交付
- Release 启用优化、Dead Code Stripping、Hardened Runtime 和产品验证。
- 发布流程升级为 Archive、Developer ID 导出、嵌入扩展与 entitlement 验证、DMG 公证、staple，并对 DMG 内最终 App 执行 Gatekeeper 验收。

### 🎨 品牌视觉
- 正式采用“光标 + 菜单 + 闪光”的方案 3 Logo，使用无渐变、无阴影的纯色扁平风格。
- 新增可维护的 SVG 矢量母版，并为 macOS App Icon 补齐 16 px 至 1024 px 的独立尺寸资源。
- README Logo 与应用图标统一使用靛蓝、米白、青色三色规范，圆角外部改为透明。

### ⚙️ 设置体验
- “关于”页的公众号、作者微信和支持入口增加原生悬浮二维码卡片，鼠标移入对应 CTA 行即可展示。
- 快速切换 CTA 或将鼠标移入弹层时保持内容与高亮项一致，移出后延迟收起，避免闪烁。
- 二维码只进入主设置 App，并使用一次性静态缓存，不增加 Finder 扩展体积或空闲任务。
- Finder 扩展的进程显示名称改为“SwiftMenu Finder 扩展”，避免在活动监视器中与设置主程序混淆为重复进程。
- “运行方式”增加“管理 Finder 扩展…”按钮，跳转系统扩展设置以正式启用或彻底停用扩展；关闭设置窗口仍只退出主程序。

## v1.1.0 (2026-02-05)

### ✨ 新增功能 (New Features)
- **增加“复制”菜单**：支持直接在右键菜单中复制文件到系统剪贴板，与其他应用完美互通。
- **菜单顺序调整**：在设置界面中支持拖拽调整右键菜单的显示顺序。
- **系统剪贴板集成**：“粘贴”功能现在支持粘贴来自系统剪贴板的文件（即在其他地方复制的文件）。
- **更广泛的系统兼容**：最低支持 macOS 12.0 (Monterey)，兼容 Apple Silicon 和 Intel 架构。
- **Release 日志**：发布脚本现在支持手动输入 Release Note。

### 🛠 优化改进 (Improvements)
- 优化了粘贴逻辑，智能处理剪切和复制的冲突（覆盖/跳过/重命名）。
- 设置界面重构为标签页结构（常规/菜单排序），更加清晰易用。
- 更新了 README 文档，增加了详细的系统要求说明。

## v1.0.0 (2026-02-03)

### 🎉 首次发布 (Initial Release)
- **新建文件**：支持 .txt, .docx, .xlsx, .pptx, .md 等常见格式。
- **复制路径**：一键复制文件绝对路径。
- **在终端打开**：在当前目录快速启动终端。
- **开机自启**：支持随系统启动自动运行。
- **自定义设置**：可配置显示的菜单项。
