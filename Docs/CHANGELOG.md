# 更新日志 (CHANGELOG)

## v1.1.0 (2026-02-05)

### ✨ 新增功能 (New Features)
- **增加“复制”菜单**：支持直接在右键菜单中复制文件到系统剪贴板，与其他应用完美互通。
- **菜单顺序调整**：在设置界面中支持拖拽调整右键菜单的显示顺序。
- **系统剪贴板集成**：“粘贴”功能现在支持粘贴来自系统剪贴板的文件（即在其他地方复制的文件）。
- **更广泛的系统兼容**：最低支持 macOS 12.0 (Monterey)，兼容 Apple Silicon 和 Intel 架构。
- **Release 日志**：发布脚本现在支持手动输入 Release Note。

### 🛠 优化改进 (Improvements)
- 优化了粘贴逻辑，智能处理剪切和复制的冲突（覆盖/跳过/重命名）。
- 设置界面重构为标签页结构（常规/菜单排序），更加清晰易用。
- 更新了 README 文档，增加了详细的系统要求说明。

## v1.0.0 (2026-02-03)

### 🎉 首次发布 (Initial Release)
- **新建文件**：支持 .txt, .docx, .xlsx, .pptx, .md 等常见格式。
- **复制路径**：一键复制文件绝对路径。
- **在终端打开**：在当前目录快速启动终端。
- **开机自启**：支持随系统启动自动运行。
- **自定义设置**：可配置显示的菜单项。
