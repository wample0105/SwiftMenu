# 设置页 CTA 悬浮交互设计

SwiftMenu 设置页的三个 CTA 使用原生 SwiftUI `popover` 展示对应二维码，交互参考阿坡系列桌面启动器的悬浮二维码入口。

## 行为

- 鼠标进入任意 CTA 行时，立即展示该行对应的二维码。
- 鼠标移出 CTA 行后延迟 160 毫秒收起，避免移动到弹层时闪退。
- 鼠标进入二维码弹层会取消关闭任务；离开行和弹层后自动收起。
- CTA 行始终显示轻量二维码图标，悬浮时使用浅色背景提示可交互性。
- README 的二维码入口继续保留，作为键盘导航和外部查看的兜底方式。

## 资源与性能

- 三张二维码只打包进主设置 App，不进入 Finder Sync 扩展。
- 二维码通过静态缓存首次按需加载，悬浮过程中不重复读取磁盘。
- 没有常驻定时器；160 毫秒延迟任务只在鼠标离开时短暂创建，并可立即取消。
