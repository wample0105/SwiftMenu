# RightMenu 快速开始

## 三步安装

### 1️⃣ 安装应用
- 打开 `RightMenu_Installer.dmg`
- 将 `RightMenu.app` 拖到"应用程序"
- **右键**打开（首次需要）

### 2️⃣ 启用扩展
- 打开 **系统设置** → **扩展** → **Finder 扩展**
- 勾选 ✅ **RightMenu**

### 3️⃣ 开始使用
- 在 Finder 任意位置**右键**
- 看到"新建..."菜单即表示成功！

---

## 🎯 主要功能

| 功能 | 说明 |
|------|------|
| 📄 新建... | 快速创建 txt/docx/xlsx/pptx/md 文件 |
| 📋 复制路径 | 一键复制文件/文件夹路径 |
| 💻 在终端打开 | 在当前目录打开终端 |

---

## ⚠️ 重要提示

- **首次打开**应用必须右键 → "打开"
- 如果菜单没出现，请重启 Finder
- 所有功能可在主应用中自定义
